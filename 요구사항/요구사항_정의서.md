# 부동산 매물 관리 시스템 - 요구사항 정의서

> 작성일: 2026-03-20
> 기반: 요구사항 이미지 2장 (매물 상세 등록 화면, 매물 목록 화면)

---

## 1. 매물 목록 페이지

### 1.1 검색/필터 영역

| 항목 | 설명 |
|------|------|
| 물건유형 필터 | 상가/점포/사무실 등 유형별 필터링 |
| 거래유형 필터 | 월세/전세/매매 등 거래 유형별 필터링 |
| 상태 필터 | 물건등록, 거래완료 등 상태별 필터링 |
| 텍스트 검색 | 주소, 물건번호 등 키워드 검색 |
| 탭 네비게이션 | 매물관리(상가) 등 카테고리별 탭 전환 |

### 1.2 매물 카드 그리드

| 항목 | 설명 |
|------|------|
| 썸네일 이미지 | 매물 대표 사진 표시 |
| 매물명/위치 | 매물 이름 및 위치 정보 |
| 가격 정보 | 월세/보증금/매매가 표시 (예: 월 71.5만) |
| 면적 정보 | 평수 표시 (예: 23평) |
| 상태 표시 | 매물 상태 아이콘/뱃지 |
| 정렬 기능 | 가격순, 면적순, 등록일순 정렬 |

---

## 2. 매물 상세 등록/수정 페이지

### 2.1 기본 정보 섹션 (좌측 패널)

| 필드 | 타입 | 설명 |
|------|------|------|
| 물건번호 | 자동생성 (읽기전용) | 예: #Y480081177 |
| 물건유형 | Select | 상가/점포/사무실 |
| 상태 | Select | 물건등록, 거래완료 등 |
| 등록일 | Date | 매물 등록 날짜 |
| 직접물건 여부 | Checkbox | 직접 물건 여부 |
| 거래유형 | Select | 월세/전세/매매 |
| 기존중개인 | Text | 기존 담당 중개인 |
| 소유자이름 | Text | 소유자 성명 |
| 매도인/임대인 | Text + Phone | 이름, 전화번호, 관계(건물주 등) |
| 매수인/임차인 | Text + Phone | 이름, 전화번호 |
| 간전매도인 | Text | 간접 매도인 정보 |
| 접수일 | Date | 매물 접수 날짜 |
| 최종확인일 | Date | 마지막 확인 날짜 |
| 작업일시 | DateTime | 작업 일시 |
| 평점 | Rating (★) | 1~5점 별점 |
| 메모 | Textarea | 자유 메모 (별칭, 특이사항 등) |

### 2.2 주소 정보 섹션

| 필드 | 타입 | 설명 |
|------|------|------|
| 지번주소 | Text + 검색 | 예: 광주 서구 천정동 97-41 화정서부시장 |
| 도로명주소 | Text | 예: 광주 서구 내방로 320 |
| 면적 (평) | Number | 평수 입력 |
| 면적 (㎡) | Number | 제곱미터 (평↔㎡ 자동 변환) |
| 지도 연동 | Button | 지도보기, 로드뷰 등 외부 연동 |
| 태그 | Multi-select | 특이사항 태그 (역세권, 코너 등) |

### 2.3 거래 정보 섹션

| 필드 | 타입 | 설명 |
|------|------|------|
| 거래유형 탭 | Tab (월세/전세/매매/교환) | 탭 선택에 따라 입력 필드 변경 |
| 보증금 | Number (만원) | 보증금액 |
| 월세금 | Number (만원) | 월 임대료 |
| 관리비 | Number (만원) | 월 관리비 |
| 관리비 포함 여부 | Radio | 관리비없음 / 관리비포함 / 별도 |
| 관리항목 | Checkbox Group | 전기, 가스, 수도, 인터넷, TV 등 |
| 입주가능일 | Date / "즉시" 토글 | 입주 가능 시점 |
| 융자금 | Number (만원) | 융자금액 |
| 권리금 | Number (만원) | 권리금액 |

### 2.4 건물 정보 섹션

| 필드 | 타입 | 설명 |
|------|------|------|
| 건물유형 | Select | 상가건물, 주상복합 등 |
| 건축물용도 | Select | 근린생활시설 등 |
| 대지면적 | Number (㎡) | 대지 면적 |
| 전용면적 | Number (㎡/평) | 전용 면적 (자동 변환) |
| 공급면적 | Number (㎡/평) | 공급 면적 (자동 변환) |
| 해당층/총층 | Number / Number | 예: 1층 / 4층 |
| 사업자등록 가능 여부 | Checkbox | 사업자등록 가능 여부 |
| 현관구조 | Select | 현관 방향/구조 |
| 주출입구 기준 | Select | 출입구 방향 |
| 사용승인일 | Date | 건축물 사용 승인 날짜 (예: 2011-09-20) |
| 층/실 | Text | 해당 층과 실 번호 |

## 3. 공통 기능 요구사항

| 기능 | 설명 |
|------|------|
| 매물 저장 | 입력한 매물 정보 저장 |
| 매물 복제 | 기존 매물을 복사하여 새 매물 생성 |
| 매물 삭제 | 일반 삭제 / 완전 삭제 구분 |
| 닫기 | 폼 닫기 (변경사항 미저장 시 확인 다이얼로그) |
| 인쇄 | 매물 정보 인쇄 기능 |
| 평/㎡ 자동 변환 | 면적 입력 시 평↔㎡ 자동 계산 |
| 전화번호 포맷 | 전화번호 자동 포맷팅 (010-XXXX-XXXX) |
| 가격 단위 표시 | 만원 단위 입력, 억 단위 자동 표시 |
| localStorage 저장 | 오프라인 환경 대응, 로컬 데이터 저장 |
